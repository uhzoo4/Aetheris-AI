# 🌌 Aetheris AI — Autonomous Data Orchestration

> **Orchestrating complex enterprise workflows with absolute mathematical integrity.**

Aetheris AI is a premium, cinematic enterprise data synthesis platform. It combines a state-of-the-art brand system (**Nocturnal Luxury**) with highly calibrated motion design, cryptographic security assurances, and production-grade frontend engineering.

## 🎬 Cinematic & Motion Features

Every interaction in Aetheris is physically calibrated to feel weighted, tactile, and deliberate:

1.  **Editorial Matrix Preloader**: An initial fullscreen clip-path curtain wipe coupled with an eased count-up from `000` to `100` that loads initial atmosphere.
2.  **Cinematic Custom Cursor**: A dual-ring cursor tracking system that reacts with fluid inertia (`lerp` factor: `0.28`) and shifts scale/color on interactive anchors.
3.  **Physical Tilt & Glint Coordination**: Cards reacting to pointer positions with subtle rotation (`maxRotation: 2.5deg`) and an coordinate-interpolated spotlight glass highlight overlay (`--mouse-x`, `--mouse-y`) that glides across glass gradients.
4.  **Magnetic Pull Controls**: High-attention buttons that dynamically pull toward the cursor, easing smoothly back to center without snapping.
5.  **Line-Masked Editorial Reveals**: Text headers that slide up from behind invisible boundary boxes when scrolled into view.

## 🛠️ Production-Ready Engineering

This application is built with clean, zero-dependency architectural utilities:

*   **Global Event-Driven Feedback Dialog**: Resolves dead button links globally via a lightweight Custom Event listener (`open-aetheris-modal`). Invoked dynamically across all action triggers (`Login`, `Request Access`, `Explore Security`, `Select Pro`, `Contact Sales`, and legal policy anchors).
*   **Mobile Hamburger Drawer**: A responsive (`md:hidden`), fully animated slide-down navigation drawer overlay that integrates perfectly with page anchors and closes gracefully on click.
*   **Smooth Scroll Navigation**: Native CSS-driven smooth scrolling coupled with exact HTML section identifier landmarks (`#features`, `#testimonials`, `#pricing`).
*   **Accessibility & UX Safety**: Respects `prefers-reduced-motion` media queries (skipping the preloader instantly), supports complete keyboard navigation overlays (`focus-visible:ring`), and enforces standard ARIA landmark specifications.

## 🚀 Getting Started

To run the codebase locally in development mode:

```bash
# Install dependencies
npm install

# Run the local Vite server
npm run dev
```

To compile a highly optimized production bundle:

```bash
# Run compiler and builder
npm run build
```

The production output compiles into a minimal, optimized package ready for zero-latency hosting:
